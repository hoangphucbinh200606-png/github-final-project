# Simple Interest Calculator

A simple bash script to calculate the simple interest based on user input.

## Formula
The formula used for the calculation is:
Simple Interest = (Principal * Rate * Time) / 100

## How to use
1. Clone this repository to your local machine.
2. Ensure the script is executable by running: `chmod +x simple-interest.sh`
3. Execute the script:
   ./simple-interest.sh
4. Follow the on-screen prompts to enter the Principal, Rate of Interest, and Time Period.
