# Contributing

All contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome.

## How to contribute
- **Bug Reports:** If you find a bug, please open an issue.
- **Bug Fixes:** Feel free to submit a pull request for any bug fixes.
- **Documentation:** Improvements to documentation are always welcome.
- **Enhancements:** If you have ideas for enhancements, please open an issue to discuss.
